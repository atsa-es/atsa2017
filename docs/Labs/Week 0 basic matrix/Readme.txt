The chapter
basic-matrix-math.Rnw is the wrapper.  Open in RStudio and use Compile pdf
basic-matrix-math-content.Rnw is the content
The book uses a tex version of 'basic-matrix-math-content.Rnw'.  To get that, switch to the directory, and run the Sweave command on that file.

The key
basic-matrix-math-key.Rnw is the wrapper.  Open in RStudio and use Compile pdf
basic-matrix-math-key-content.Rnw is the content
The book uses a tex version of 'basic-matrix-math-key-content.Rnw' since we don't want the header info. To get that, switch to the directory, and run the Sweave command on that file.
